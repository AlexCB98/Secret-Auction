# Secret Auction

A simple secret auction program built in Python.

## Features

- Multiple users can place bids;
- Stores bids using a dictionary;
- Determines the highest bidder automatically;
- Clears screen between users (simulated);
- Simple command-line interface.

## How it works

1. Each user enters their name and bid;
2. The program stores all bids in a dictionary;
3. Users take turns without seeing previous bids;
4. At the end, the highest bidder is selected.

## Project structure

- `main.py` → main program logic

## What I practiced

- dictionaries;
- loops;
- functions;
- conditionals;
- user input handling.

## How to run

Run the program using Python:

```
python main.py
```
or run directly in PyCharm.
